Kaip pakeisti WiFi slaptažodį

Universali instrukcija WiFi slaptažodžio keitimui per maršrutizatoriaus nustatymus — tinka visiems populiariausiems routeriams.

Atnaujinta: 2026

Skaidrumas ir pasitikėjimas

Kai kurios nuorodos šiame puslapyje yra affiliate partnerių nuorodos. Tai reiškia, kad galime gauti komisinį mokestį, jei nuspręsite įsigyti produktą per mūsų nuorodą — jums tai nekainuoja papildomai. Mūsų apžvalgos ir rekomendacijos yra nepriklausomos ir pagrįstos objektyviais kriterijais.

Turinys

  1. 01 1. Prisijunkite prie routerio
  2. 02 2. Raskite WiFi nustatymus
  3. 03 3. Pakeiskite slaptažodį
  4. 04 4. Prijunkite įrenginius iš naujo
  5. 05 Nustatymai pagal gamintoją

1. Prisijunkite prie routerio

Atidarykite naršyklę ir įveskite routerio IP adresą (dažniausiai 192.168.0.1 arba 192.168.1.1). Prisijunkite su admin vartotoju — slaptažodis paprastai nurodytas ant routerio lipduko apačioje.

2. Raskite WiFi nustatymus

Ieškokite skyriaus „Wireless", „WiFi Settings" arba „Belaidis tinklas". Kiekvieno gamintojo sąsaja šiek tiek skiriasi, bet WiFi nustatymai visada yra pagrindinėse kategorijose.

3. Pakeiskite slaptažodį

Laukelyje „Password", „Pre-Shared Key" arba „Passphrase" įveskite naują slaptažodį. Rekomenduojama: WPA3 arba WPA2 šifravimas, mažiausiai 12 simbolių. Išsaugokite nustatymus.

4. Prijunkite įrenginius iš naujo

Pakeitę WiFi slaptažodį, visi prijungti įrenginiai bus atjungti. Kiekviename įrenginyje pasirinkite savo WiFi tinklą ir įveskite naują slaptažodį.

Nustatymai pagal gamintoją

TP-Link: 192.168.0.1 → Wireless → Security. ASUS: router.asus.com → Advanced Settings → Wireless. Netgear: routerlogin.net → Wireless. Huawei: 192.168.3.1 → My WiFi. Telia/Tele2: dažniausiai 192.168.1.1, admin/admin.

Dažnai užduodami klausimai

Kaip sužinoti routerio IP adresą?

Windows: atidarykite Command Prompt ir įveskite „ipconfig" — ieškokite „Default Gateway". Mac: System Preferences → Network → Advanced → TCP/IP.

Kaip pakeisti WiFi slaptažodį telefone?

Atidarykite telefono naršyklę ir įveskite routerio IP adresą. Prisijunkite kaip admin ir keiskite slaptažodį lygiai taip pat kaip kompiuteryje.

Kokį WiFi šifravimą pasirinkti?

Visada rinkitės WPA3 (jei palaiko routeris) arba WPA2-PSK (AES). Venkite WEP ir WPA — jie nesaugūs.
